Pokedex

Desarrollé una aplicación Pokedex utilizando React, donde los usuarios pueden buscar y visualizar información detallada sobre diferentes Pokémon. Implementé componentes reutilizables y manejé estados para una experiencia de usuario interactiva y fluida.

Rick and Morty App

Creé una aplicación en React que permite explorar personajes, episodios y ubicaciones de la serie Rick and Morty. Utilicé la API oficial para integrar datos dinámicos y proporcioné una interfaz intuitiva con componentes modulares y gestión de estados eficiente.

Weather App

Desarrollé una aplicación del clima en React que muestra información meteorológica en tiempo real para cualquier ubicación. Utilicé APIs de clima para obtener datos precisos, implementando una interfaz limpia y amigable, con gráficos y detalles meteorológicos como temperatura, humedad y nubosidad.

Fortune Cookies

Desarrollé una aplicación en React que presenta frases inspiradoras de personas famosas, simulando galletas de la fortuna. Utilicé un archivo JSON como fuente de datos, implementando componentes interactivos para mostrar una nueva frase al hacer clic, proporcionando una experiencia divertida y motivadora para los usuarios.
